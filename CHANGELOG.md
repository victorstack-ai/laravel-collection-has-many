# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-07

### Added

- `hasMany()` macro for Laravel Collections with O(n+m) performance.
- Support for both array and object parent items.
- Automatic conversion of plain arrays to collections for related items.
- Customizable local key, foreign key, and relation name parameters.
- Full PHPUnit test coverage.
